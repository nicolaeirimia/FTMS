Freight Transport Management System (FTMS)

Manage shipments, fleet, and customer relationships efficiently.

Features

  • Shipment Management
  • Fleet and Resource Management
  • Customer Management and Billing

Technology Stack

  • Backend: Java + Spring Boot
  • Frontend: React SPA
  • Database: PostgreSQL
  • Object Storage: S3
  • Message Queue: RabbitMQ/Kafka
